# Bullet points

What the OMOP CDM gives us:

- **Standardised structure** — every site stores data in the same tables
- **Shared vocabularies** — concepts mapped to OMOP standard codes
- **Portable analytics** — the same query runs at every participating site
- **Privacy by design** — patient-level data never leaves the institution
