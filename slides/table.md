# Table layout

OMOP CDM core clinical tables:

| Table | Holds | Example concept |
|-------|-------|-----------------|
| `person` | Demographics, one row per patient | Gender, year of birth |
| `condition_occurrence` | Diagnoses | Type 2 diabetes |
| `drug_exposure` | Prescriptions & dispensing | Metformin |
| `measurement` | Labs & vitals | HbA1c, blood pressure |
| `visit_occurrence` | Encounters | Inpatient stay |

Replace rows with your own content — the theme styles tables automatically.
